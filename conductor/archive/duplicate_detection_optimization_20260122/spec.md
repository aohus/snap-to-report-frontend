# Specification - 중복 사진 제거 성능 고도화

## 1. 개요
사용자가 업로드하기 위해 선택한 사진들 중에서 중복되거나 시각적으로 거의 동일한 사진을 빠르게 찾아내어 제외할 수 있도록 돕는 기능을 고도화합니다.

## 2. 목표
- **속도 최적화:** 500장 이상의 사진 중복 검사를 5초 이내에 완료 (Web Worker 활용).
- **정확도 향상:** 파일명 일치(Pass 1)와 시각적 유사성(Pass 2) 검사 로직을 정교화.
- **UI 개선:** 검사 중임을 알리는 상태 표시 및 직관적인 결과 리스트 제공.

## 3. 상세 기능
- **Web Worker 기반 검사:** 메인 스레드 부하 없이 백그라운드에서 해싱 및 비교 수행.
- **시각적 해싱(Perceptual Hashing):** 이미지의 픽셀 데이터를 분석하여 크기나 포맷이 달라도 내용이 같은 사진을 감지.
- **메모리 최적화:** 대량의 이미지 데이터를 처리할 때 메모리 누수 방지 및 효율적인 리소스 관리.

## 4. 기술적 고려 사항
- 기존 `duplicateDetection.ts` 로직을 Worker 환경으로 이식.
- Canvas API 대신 Worker에서 사용 가능한 `createImageBitmap` 적극 활용.
